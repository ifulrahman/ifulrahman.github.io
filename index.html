  • Redux

    Redux is a state management library for JavaScript applications, often used with React. It provides a centralized store to manage the state of an entire application, making it predictable and easier to debug. With Redux, components can access and update the state without passing data through multiple layers, reducing complexity in large applications. It follows a unidirectional data flow, using actions to describe state changes and reducers to handle updates, promoting maintainable and scalable code.

The Raspberry Pi 4 Model B serves as the main microcontroller running the system, with the Raspbian operating system as its platform.
