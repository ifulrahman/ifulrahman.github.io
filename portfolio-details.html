Description

A Face and Temperature Detection System for Attendance and Body Temperature Checking Based on IoT is a device that combines Internet of Things (IoT) technology with facial detection and body temperature measurement. This device utilizes hardware such as a webcam for facial detection and an AMG8833 temperature sensor for measuring body temperature. The Raspberry Pi 4 Model B serves as the main microcontroller running the system, with the Raspbian operating system as its platform. For facial detection, this device leverages the Local Binary Pattern Histogram and Haar Cascade Classifier algorithms, supported by an intuitive graphical user interface (GUI). Users can easily issue facial detection commands and capture facial samples through this GUI.
